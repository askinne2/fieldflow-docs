## Profile

Go to **Account** in the sidebar to update:
- **First name** and **Last name**
- **Email** — displayed but not editable (tied to your authentication)

## Password

By default, FieldFlow uses email verification codes for sign-in. You can optionally set a password for faster access:

1. Go to **Account**
2. Click **Set Password** (or **Change Password** if one is already set)
3. Follow the verification flow

Even with a password set, email code sign-in remains available as a fallback.

## Workspace preferences

Go to **Settings** in the sidebar to configure:

| Setting | Options |
|---|---|
| **Default project** | Which project loads first on the dashboard |
| **Timezone** | Auto-detect or manual selection |
| **Date format** | US (MM/DD/YYYY) or International (DD/MM/YYYY) |
| **Items per page** | 10, 25, 50, or 100 (for dashboard lists) |

## Notifications

Control which emails you receive:

| Notification | Default | Description |
|---|---|---|
| **Sync failure alerts** | On | Email when a sync fails after all retries |
| **Product updates** | On | Occasional emails about new features |
| **Weekly digest** | Coming soon | Summary of sync activity and usage |

## Danger zone

At the bottom of Settings:
- **Export data** — download all your project data as JSON
- **Delete account** — permanently removes your account, projects, and data. Requires email confirmation. This cannot be undone.
