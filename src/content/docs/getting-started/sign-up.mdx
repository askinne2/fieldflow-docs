## Create your account

1. Go to [app.getfieldflow.io](https://app.getfieldflow.io) and click **Get Started Free**
2. Enter your email address
3. Check your inbox for a verification code and enter it
4. You're in — no credit card required

FieldFlow uses email-based authentication. You'll receive a one-time code each time you sign in. If you prefer, you can set a password later in **Account Settings**.

## Your first dashboard visit

After signing in, you'll land on the **Dashboard**. If this is your first time, you'll see an onboarding checklist that walks you through:

1. Connect Google Sheets
2. Create your first project
3. Map your fields
4. Customize the layout
5. Publish your directory

Each step links directly to the right place in the app.
