---
title: Sharing Links
description: Share your directory, quiz, and detail page URLs directly.
---

Every FieldFlow project generates shareable URLs you can send to anyone — no embed code needed.

## Directory URL

Your public directory lives at:

```
https://app.getfieldflow.io/d/[your-project-id]
```

Find this URL in the **Publish** tab under **Share**. Click **Copy** to copy it to your clipboard, or **Open** to view it in a new tab.

This is a full-featured directory with search, filters, and pagination. Anyone with the link can view it.

## Quiz URL

If you have Quiz Flow enabled, a quiz link is also available:

```
https://app.getfieldflow.io/d/[your-project-id]/quiz
```

This starts visitors with the quiz, then shows filtered results.

## Detail page URLs

If detail pages are enabled, each record has its own URL:

```
https://app.getfieldflow.io/p/[your-project-id]/[record-id]
```

These URLs are generated automatically. Visitors reach them by clicking a card in the directory. You can also share individual record URLs directly.

## When to use sharing links vs. embeds

| Use case | Best option |
|---|---|
| Quick share via email, Slack, or social media | Sharing link |
| Embed on your website so visitors never leave | [Embed code](/publishing/embed-code/) |
| Link from a newsletter | Sharing link |
| Add to a navigation menu on your site | Either works |

Sharing links open the directory on `app.getfieldflow.io`. Embeds display the directory directly on your website.
